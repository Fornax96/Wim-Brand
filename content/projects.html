Dit zijn een paar projecten waar ik aan heb gewerkt in mijn vrije tijd:

  • Pixeldrain
    Een programma wat ik geschreven heb om schreenshots en bestanden snel en makkelijk te delen. Pixeldrain was mijn eerste echte project in Java, ik heb veel geleerd tijdens het ontwikkelen.
  • Radio FORNAX
    Een online radiostation wat muziek speelt van YouTube. Als een gebruiker een nummer aanvraagt wordt op de server de muziek gedownload van YouTube met gebruik van de VLCj library, vervolgens wordt het in een afspeellijst gezet en wordt gestreamed naar alle luisteraars. De server side is geprogammeerd in Java met behulp van Apache Tomcat, ik wil graag nog meer projecten maken met Apache Tomcat omdat het zo leerzaam is.
    (Muziek aanvragen is op dit moment niet mogelijk)
    (De website draait op mijn eigen PC thuis, en is niet altijd bereikbaar)
